# Clone Disney+ 🎬

Recriação fiel da página inicial do Disney+, desenvolvida como projeto de estudo de front-end. A página conta com hero section com planos de assinatura, listagem de séries por abas (Em Breve, Mais Populares, Mais no Star+), seção de planos, dispositivos compatíveis, FAQ interativo e footer completo.

## 🛠 Tecnologias utilizadas

- HTML5
- SCSS (compilado com Gulp)
- JavaScript
- Gulp (task runner)

## 🚀 Como rodar o projeto

```bash
npm install
npx gulp
```
